# Tic-Tac-Toe-on-calculator
A simple and fun Tic Tac Toe game designed to run on a Casio Graph Math+ calculator.

## Description
This project is a Tic Tac Toe with a graphic interface, adapted to be played on a Casio Graph Math+ calculator. The goal is to line 3 of our symbol (`X` or `O`).

## Features
- **Players:** Choose the nickname of each player to play.
- **Play:** Use the number keys which represent the interface. The 1st player has the cross and the 2nde teh circle.
- **Restart:** You can replay without restaring the file.
- **See the score:** When you finish to play you can see the score and who win the more rounds.

## Requirements
A **Casio Graph Math+ calculator.**

Required libraries to run Python scripts on the calculator: **casioplot**, **turtle** and **random**.

## Installation
1. Download the files from this repository.
2. Transfer them to your calculator.
3. Run the morpturt.py file to start the game.

## How to Play
- Start the Game: Enter the nicknames to play.
- Play: Press any `number key` to place your symbols.
- Replay: Play again for more fun.
- Or stop playing : And see who win.
